学习使用gulp-jade模板引擎

Jade是一个基于javascript，可以运行在node与浏览器环境的一个前端模板引擎，

对于日益庞大的前端页面来说，还是用老的字符串拼接的方法来嵌入html内容已经

不适应当下的开发趋势了。

1、安装gulp-jade

1、jade基于nodejs的开发环境，所以我们首先要安装nodejs,nodejs的安装方法

其实在之前的博客中已经提及了，这里就不在赘述了，[nodejs官网](https://nodejs.org/en/)可以直接下载安装包。

2、安装gulp-jade插件，命令：$ npm install -g --save-dev gulp-jade

如果需要权限就在npm之前加上sudo。
